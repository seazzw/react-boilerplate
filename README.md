# 从零开始的第一个react项目
基于 React16.x  + Webpack4.x + Babel7.x  的模板库.

## 安装

```bash
git clone --depth=1  https://github.com/seazzw/react-boilerplate.git  <你的项目名字>
cd  <你的项目名字>
```
使用 NPM 安装依赖:

```bash
npm install
```

或者使用 YARN 安装依赖:

```bash
yarn install
```
NPM Scripts 说明:

|`npm run <script>` |Description|
|-------------------|-----------|
|`start`            |启动服务 `localhost:3000`|
|`build`            |打包项目|
|`test`             |执行测试|
|`lint`             |检查文件格式|
|`lint:fix`         |检查文件格式并自动格式化|
